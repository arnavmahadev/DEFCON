“Better prevent than tackle” holds up, directionally, on a World Cup pilot
I ran the full credit pipeline over 7 knockout games (34 players) of PFF’s 2022 World Cup tracking, joined Transfermarkt market values (Nov 2022), and correlated each credit type against value. For defenders, the paper’s core pattern shows up: value tracks positioning, not ball-winning.
RAW INTERCEPT vs VALUE
r = +0.02
ball-winning barely tracks value
PREVENTION (DISTURB) vs VALUE
r = +0.51
positioning strongly does
Caveats: the EPV models are trained on Metrica (cross-provider domain shift, so magnitudes approximate), n = 34 over 7 games (not the paper’s 564), so this is directional, not statistically significant. The concede-negative half of the paper’s claim does not cleanly reproduce at this sample size.
